# Less Annoying CRM API Documentation - Introduction

## Overview
The Less Annoying CRM API enables external programs to interact with the platform. Rather than providing a user interface for human interaction, the API allows developers to programmatically read and write information to the CRM system.

## Key Points
- The API is described as "an advanced feature" requiring capable programmer expertise
- The platform emphasizes simplicity in its API design philosophy
- Users unfamiliar with programming concepts like APIs are directed back to the main CRM interface

## Purpose Statement
The documentation states: "Our API lets programmers write code that enters and retrieves information to/from the CRM."

## Disclaimer
The documentation notes that support staff cannot write or debug code, but remain available to discuss API implementation strategies.
