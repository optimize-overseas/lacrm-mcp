# Less Annoying CRM API Documentation - Pipeline Icons

## Overview
The Less Annoying CRM API allows you to select icons when creating and editing pipelines. The icons are referenced by their name identifiers when making API calls.

---

## Usage

When creating or editing a pipeline via the API, use the icon name (snake_case format) in the `Icon` parameter:

```json
{
  "Function": "CreatePipeline",
  "Parameters": {
    "Name": "Sales Pipeline",
    "Icon": "dollar_sign",
    "Permissions": "Public"
  }
}
```
